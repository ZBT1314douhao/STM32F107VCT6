  1. 微氧测量仪模拟设计(通过STM32F107VCT6片内外设模拟各功能)
    硬件平台：STM32F107VCT6开发板
    软件平台：STM32Cube MX 4.16 & MDK V5.20
    触摸屏驱动方式：SPI通讯（SPI3）
    显示界面设计：STemwin V5.22
